How-to

Install, launch, and keep Teams green.

Follow these steps to get Available On Teams running. Once it is in your tray, it will gently move the mouse so Microsoft Teams never flips you to “Away”.

Download setup View on GitHub

Windows only. You can allow it to start with Windows or launch it manually whenever you need the green light.

Install & set up

  1. Download the latest installer from the releases page.
  2. Run the setup and follow the prompts. Choose whether to launch Available On Teams automatically when Windows starts.
  3. Finish installation and start the app. It minimizes to your tray immediately to stay out of your way.

The app will center your cursor and perform subtle movements and clicks on a short interval, keeping Teams in Available without interrupting your work.

Control from the tray

Right-click the tray icon to start or stop. When it is running you will see a green status to confirm it is active.

Use keyboard shortcuts

Toggle without opening the menu: Ctrl + Alt + A to start and Ctrl + Alt + S to stop.

Lightweight by design

No configuration required. Install once, run it from the tray, and let it keep your presence green while you focus.

Daily use

  1. Launch the app (or let it start with Windows).
  2. Use the tray menu or shortcuts to start/stop activity.
  3. Need help or updates? Check the GitHub repository.
