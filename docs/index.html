How it works

  1. Download the latest release and install it like any desktop app.
  2. Launch Available On Teams — it hides in your tray right away.
  3. Let it simulate subtle, periodic input so Teams thinks you are still at your desk.

Pause or resume anytime from the tray, or use the shortcuts: Ctrl + Alt + A to start and Ctrl + Alt + S to stop.
